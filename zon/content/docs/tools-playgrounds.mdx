Explore ZON with interactive tools.

## Online Playground

Try ZON directly in your browser with our interactive playground:

- **[ZON Playground](/playground)** - Convert JSON to ZON, see token savings in real-time, and experiment with different encoding modes.

## CLI Tools

The ZON CLI provides command-line utilities for working with ZON files:

- Convert JSON files to ZON format
- Validate ZON syntax
- Pretty-print ZON output

See the [CLI Guide](/docs/cli-guide) for installation and usage.

## IDE Support

- **VS Code** - Syntax highlighting for `.zonf` files (coming soon)
- **JetBrains IDEs** - Plugin support planned

## Integration Libraries

Official libraries for integrating ZON in your projects:

- [TypeScript/JavaScript](/docs/typescript) - Full encoder/decoder with streaming support
- [Python](https://github.com/ZON-Format/zon) - Reference implementation

See [Integrations](/docs/integrations) for more options.
